# ustPlayer 配布文档
中文的、面向多样音声合成编辑器工程文件的可视化工具。

截止至 **2026.01.31** ，最新版本为 **ustPlayer-v26a31**

## 下载说明
请前往 Release 页面下载最新的.exe版本。
> 我们将不再提供 `.msi` 安装包，暂无计划提供 `.app` 格式程序。

---

## 更新内容（较于比上一版本）
- 修复了由多线程窗口冲突导致的闪退问题，**因此建议您不再使用 v26a24 版本**
- 新增支持保存 uPl 专用工程文件，它们以 `.uplr` 为后缀。
- 新增 ust 音高线展示功能
- 完善用户协议及错误码字典
- 优化多项细节使用体验
